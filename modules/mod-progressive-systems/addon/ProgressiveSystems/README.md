### Installation:
1. Copy `modules/mod-progressive-systems/addon/ProgressiveSystems/` to `Interface/AddOns/`
2. Restart WoW client
3. Addon loads automatically

### Commands:
- `/ps` or `/progressive` - Open main window
- `/ps ui` - Open main window
- `/ps paragon` or `/ps pg` - Open paragon window
- `/ps stats` - Open stats window
- `/ps power` - Show power level
- `/ps info` - Show system info
- `/ps config` - Open settings

### Windows:
1. **Main Window** - Tabbed interface with:
   - Progression tab (power level, tier, points, kills, prestige)
   - Paragon tab (level, tier, points, experience bar)
   - Stats tab (all custom stats)

2. **Paragon Window** - Standalone paragon system UI
3. **Stats Window** - Standalone custom stats display
4. **Settings Window** - Addon configuration
