# Progressive Systems Addon

## Installation

1. Copy the `ProgressiveSystems` folder to your WoW AddOns directory:
   - **Windows:** `World of Warcraft/Interface/AddOns/ProgressiveSystems/`
   - **Mac:** `World of Warcraft/Interface/AddOns/ProgressiveSystems/`

2. Restart World of Warcraft

3. Enable the addon in the character select screen (click "AddOns" button)

4. Log in and type `/ps toggle` to show the UI

## Features

- **Real-time Progression Tracking** - See your points, kills, and tier in real-time
- **Difficulty Indicator** - Know your current difficulty tier at a glance
- **Progress Bar** - Visual representation of your tier progression
- **Customizable** - Move and scale the UI to your preference
- **Minimap Integration** - Quick access via minimap button (coming soon)

## Commands

- `/ps` or `/progressive` - Show help
- `/ps toggle` - Show/hide the UI
- `/ps config` - Open settings (coming soon)
- `/ps reset` - Reset UI position to center

## Configuration

The addon saves your preferences automatically:
- UI position
- UI scale
- Visibility settings

## Troubleshooting

**Addon not showing:**
- Make sure it's enabled in character select screen
- Check that files are in the correct location
- Type `/ps toggle` to manually show it

**Data not updating:**
- Make sure the server module is installed and running
- Check server logs for addon communication errors
- Try reloading UI (`/reload`)
